      • @Deprecated void dependency(String nameOrGa, String configuration, String... configurations)

        Declares a dependency on target configuration(s). Dependencies must be declared using the config.dependencies block before using this method.

        Parameters:
        nameOrGa - the name or the groupId:artifactId of the dependency found in config.dependencies.
        configuration - the target configuration, e.g, api.
        configurations - additional configurations (if any).

      • @Deprecated void enforcedPlatform(Object notation, Action<? super Dependency> action)

        Declares and configures a dependency on an enforced platform. Valid values for notation are:

        • a java.lang.CharSequence with format groupId:artifactId:version
        • a java.util.Map with group, artifactId, version keys
        • a java.lang.CharSequence with the name of a platform declared in config.dependencies
        The following configurations will be used by default:
        • api
        • implementation
        • annotationProcessor
        • testImplementation
        • testAnnotationProcessor
        • compileOnly
        • testCompileOnly
        • runtimeOnly
        • testRuntimeOnly
        Parameters:
        notation - - the coordinates of the platform
        action - the dependency configuration block
